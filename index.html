ShapeDiver — Parametric Box

Interactive parametric model embedded via ShapeDiver. Use the controls inside the viewer to adjust parameters; the model fills the frame and updates in real time.

Three.js — JSON Model Viewer

This viewer loads a Three.js JSON scene (2-web-interops/triceratops.json) using THREE.ObjectLoader. Orbit with the mouse, pan with right-drag, and scroll to zoom. A subtle grid and lights provide orientation.

Map — 3D Extrusions (GeoJSON)

This map renders polygons from 2-web-interops/hia-mapbox.geojson as 3D extrusions using the height property. Styled with a dark palette and interactive tilt/rotate controls.
